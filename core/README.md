# Core Patterns (`core/`)

Welcome to the `core/` directory. This folder contains **canonical, machine-readable architectural patterns (recipes)**.

## 🎯 Purpose

The `core/` directory is designed for **coding agents** (such as Gemini CLI, Claude Code, and Antigravity) to read and compose. 

These recipes teach fundamental patterns (such as OAuth flows, session memory, or guardrails) rather than domain-specific demos.

## 🤝 Contributing

This directory is strictly owned and maintained by the DevRel team. If you are a partner team looking to showcase a deployable sample, please contribute to the [`contrib/`](../contrib/README.md) directory.
